CSharp-Database
===============
Implemeted and Published by : Nahom Mehari-Yemane

Aim and Objectives
==================

#: To calculate employee wage and input, edit and delete entries from C# form to access database. The project also retrieves entries from database and display in drop down menu. The project enters data from the form to the access database using SQL. The Microsoft access database is connected using OleDbConnection and data entry and editing is stored in classes which are connected to the project.
